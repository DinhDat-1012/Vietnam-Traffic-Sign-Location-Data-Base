# Vietnam Traffic Sign Database

This repository contains a large traffic sign database for Vietnam.

The database includes the location and classification information of more than 60,000 traffic signs collected across Vietnam.

## Included Traffic Sign Types

The dataset contains many types of Vietnamese traffic signs, including:

* No Entry Sign
* Speed Limit Sign
* End of Restriction Sign
* Stop Sign
* And many other traffic sign categories

## Dataset Information

The database includes:

* Geographic coordinates (latitude and longitude)
* Traffic sign type/classification
* Mapillary sign identifiers
* Detection metadata

## Data Source

The traffic signs were collected and processed from Mapillary traffic sign detections.

## Usage

This dataset can be used for:

* Autonomous driving research
* HD map generation
* Traffic sign recognition AI training
* GIS and mapping applications
* Smart transportation systems
* Navigation systems

## File Format

The dataset is available in:

* PostgreSQL database
* SQL dump
* CSV export

## Coverage

The dataset covers traffic signs across many regions of Vietnam.
